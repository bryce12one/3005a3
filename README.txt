COMP3005 Assignment 3 Question 1 - Student Database Application

This application is a simple Python script that demonstrates basic CRUD (Create, Read, Update, Delete) operations on a PostgreSQL database table called students.

Requirements
- Python 3
- PostgreSQL
- psycopg2 library


Setup
- Install PostgreSQL and create a database called school.
- Use the provided SQL script to create the students table and populate it with initial data.
- Install the psycopg2 library:
- Copy code
- pip install psycopg2


Configuration
- Edit the dbname, user, password, and host variables in the script to match your PostgreSQL database credentials.

Running the Application
- Open a terminal or command prompt.
- Navigate to the directory containing the Python script.
- Run the script:
- Copy code
    python3 3005a3q1.py
- The script will perform a series of database operations and print the results to the console.
- Can comment and uncomment code as well as alter values as you see fit.
